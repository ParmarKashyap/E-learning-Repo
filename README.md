# E-Learning System

This is an E-Learning Web Application project developed using PHP, HTML, CSS, JavaScript, and MySQL.

## Features
- Student login and registration
- Teacher dashboard
- Admin panel
- Course management
- Video upload
- Study material upload
- Assignment management
- Notices and analytics

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap

## Project Purpose
This project is developed for educational and college project purposes.
